# Copy y Textos del Sitio

> El copy es parte del diseño. Un hero con texto genérico arruina el mejor diseño visual.
> Todos los textos se derivan del **Brief interno** construido en `02-briefing.md`.

---

## Principios

- **Español argentino** siempre: "comprá", "elegí", "consultá" — no "compra", "elige", "consulta"
- **Vos** como trato por defecto — cambiar a "usted" solo si el brief indica público formal/corporativo
- **Sin exclamaciones vacías**: "¡Bienvenidos!" no dice nada. El headline tiene que comunicar valor
- **Sin gerundios de apertura**: no "Ofreciendo calidad desde 2010" → sí "Calidad artesanal desde 2010"
- **Número concreto > adjetivo vago**: "Más de 200 productos" > "Gran variedad de productos"

## Hero — estructura y fórmulas

El hero tiene 3 partes: **Headline + Subtítulo + CTA**. No más.

### Fórmulas de headline por tono

| Tono del brief | Fórmula | Ejemplo |
|---|---|---|
| Cercano / informal | [Qué hacés] para [para quién] | "Ropa sustentable para mujeres reales" |
| Premium / aspiracional | [Adjetivo emocional] + [producto/servicio] | "Joyería que te representa" |
| Urgencia / promocional | [Beneficio concreto] + [acción] | "Hasta 40% off en toda la colección" |
| Técnico / profesional | [Solución] para [problema específico] | "Instalaciones eléctricas certificadas para tu industria" |
| Local / barrial | [Proximidad] + [propuesta] | "La verdulería del barrio con delivery en 30 min" |
| Minimalista | Solo el nombre + [categoría] | "MARA · Indumentaria femenina" |

### Subtítulo
1-2 oraciones que amplifican el headline. Incluye: diferencial + llamada implícita a la acción.

```
❌ "Somos una empresa dedicada a la venta de productos de calidad."
✅ "Hacemos ropa de algodón orgánico 100% argentino. Sin intermediarios, directo de nuestro taller."
```

### CTA según plan

| Plan | CTA primario | CTA secundario |
|---|---|---|
| Esencial | "Consultá por WhatsApp" | "Ver catálogo" |
| Emprendimiento | "Comprar ahora" / "Ver catálogo" | "Consultá por WhatsApp" |
| Empresa | "Comprar ahora" | "Ver ofertas" |

## Descripciones de producto

### Estructura (máx. 3 párrafos cortos)

1. **Qué es + beneficio principal** (1 oración)
2. **Características clave** (lista de 3-4 bullets o párrafo corto)
3. **Llamada a la acción implícita** (1 oración — solo en Plan Esencial)

### Por tipo de rubro

**Indumentaria / Moda**
```
[Material + sensación] + [Fit/silueta] + [Ocasión de uso]

Ejemplo: "Remera de algodón peinado 180g que no se deforma al lavar.
Corte recto, ideal para usar sola o entallada bajo saco.
Disponible en 6 colores, talle S al XXL."
```

**Gastronomía / Alimentos**
```
[Ingredientes clave / origen] + [Proceso / artesanal?] + [Cómo se consume]

Ejemplo: "Mermelada de ciruela casera elaborada con ciruelas del Alto Valle.
Sin conservantes ni colorantes. Ideal para tostadas, medialunas o relleno de facturas."
```

**Tecnología / Electrónica**
```
[Problema que resuelve] + [Especificaciones relevantes] + [Compatibilidad]

Ejemplo: "Auriculares inalámbricos con cancelación de ruido activa para trabajo remoto.
40hs de batería, micrófono omnidireccional, conectividad Bluetooth 5.3.
Compatible con Mac, Windows y smartphones."
```

**Servicios / Consultoría**
```
[Resultado que logra el cliente] + [Cómo] + [Para quién]

Ejemplo: "Tu local listo para abrir en 72 horas.
Hacemos la habilitación municipal, planos y certificados por vos.
Para emprendedores y comercios en CABA y GBA."
```

**Salud / Belleza**
```
[Beneficio específico] + [Ingredientes o técnica] + [Resultado esperado]

Ejemplo: "Crema hidratante con ácido hialurónico y vitamina C.
Textura gel-crema de rápida absorción, sin parabenos.
Piel visiblemente más luminosa desde la primera semana."
```

## Página "Sobre nosotros"

Estructura en 4 bloques (no más de 300 palabras totales):

1. **Origen** — cómo y por qué nació el negocio (contar la historia real, no genérica)
2. **Propósito** — qué problema resuelven, para quién
3. **Diferencial** — por qué elegirlos vs la competencia (concreto, sin adjetivos vacíos)
4. **Equipo / Cara humana** — nombre(s), ciudad, dato personal si el brief lo permite

```
❌ "Somos una empresa comprometida con la excelencia y la satisfacción del cliente."
✅ "Empezamos en 2019 en el garage de casa, haciendo sahumerios a mano para regalar.
   Hoy despachamos a todo el país con los mismos ingredientes y el mismo cuidado."
```

## FAQ — preguntas sugeridas por rubro

Siempre incluir las 3 primeras. Agregar las específicas según el brief:

**Universales (todos los rubros)**
- ¿Cómo puedo hacer un pedido?
- ¿Cuáles son los medios de pago? *(adaptar según plan)*
- ¿Hacen envíos a todo el país?

**Indumentaria**
- ¿Cómo sé qué talle elegir?
- ¿Puedo cambiar o devolver si no me queda bien?

**Alimentos / Gastronomía**
- ¿Tienen fecha de vencimiento? ¿Cómo conservarlos?
- ¿Los productos tienen certificación o habilitación bromatológica?

**Servicios**
- ¿Cómo es el proceso de contratación?
- ¿Trabajan en toda Argentina o solo en una zona?

**E-commerce general**
- ¿Cuánto tarda en llegar mi pedido?
- ¿Puedo rastrear mi envío?
- ¿Cómo aplico un cupón de descuento?

## Textos de interfaz (microcopy)

Consistencia en toda la UI:

| Elemento | Texto correcto | Evitar |
|---|---|---|
| Botón agregar carrito | "Agregar al carrito" | "Add to cart", "Añadir" |
| Botón comprar | "Comprar ahora" | "Buy now", "Proceder" |
| Stock agotado | "Sin stock" | "Out of stock", "Agotado" |
| Stock bajo | "Últimas X unidades" | "Low stock", "Pocas unidades disponibles" |
| Cargando | "Cargando…" | "Loading…", spinner sin texto |
| Error de formulario | "Ingresá un email válido" | "Invalid email", "Error en el campo" |
| Envío gratis | "Envío sin cargo" | "Free shipping", "Envío gratis" |
| Precio tachado | "$X · Antes $Y" | "Sale", "Descuento" |
| Vacío en carrito | "Tu carrito está vacío. ¡Empezá a llenar!" | "No items" |
| Pago exitoso | "¡Tu pedido fue confirmado!" | "Payment successful" |

## Textos SEO — diferencia entre copy visible y metadata

El copy visible puede ser emocional. La metadata debe ser descriptiva y contener keywords.

```
Título visible en hero:    "Ropa que te hace sentir vos"
<title> SEO:               "Remeras y camperas de algodón orgánico | Marca · Buenos Aires"

Descripción de producto:   "La remera que no se deforma ni pierde el color."
<meta description>:        "Remera de algodón peinado 180g, corte recto, 6 colores. Talle S al XXL. Envío a todo Argentina."
```

## Anti-patrones — detectar y reemplazar

| Texto genérico (AI slop) | Reemplazo específico |
|---|---|
| "productos de alta calidad" | "remeras de algodón peinado que no se encojen" |
| "excelente atención al cliente" | "respondemos por WhatsApp en menos de 2 horas" |
| "amplia variedad de productos" | "más de 150 modelos en stock permanente" |
| "los mejores precios del mercado" | "precio directo de fábrica, sin intermediarios" |
| "soluciones innovadoras" | [descripción concreta de lo que hacen] |
| "comprometidos con la excelencia" | [eliminar, no agrega valor] |
| "¡Bienvenidos a nuestra tienda!" | [eliminar, el hero habla del cliente, no de la empresa] |
